Installable through pip, once the virtualenv project has been set up.  
Activate virtualenv project  
then  
pip install -U git+https://consumers-checkbook@dev.azure.com/consumers-checkbook/DataEngine/_git/DataEngine@master  
or  
pip install --force-reinstall -U git+https://consumers-checkbook@dev.azure.com/consumers-checkbook/DataEngine/_git/DataEngine@master  
  
Its also a good idea to initiate a pull request from working to master, just in case this was overlooked.
